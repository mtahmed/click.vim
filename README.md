# click.vim

Vim syntax highlighting for click files (.testie, .template, .click).

# Installation

## pathogen

```sh
git clone git@github.com:mtahmed/click.vim.git ~/.vim/bundle/
```

## Vundle

Add this to your `.vimrc`:

```vim
Plugin 'mtahmed/click.vim'
```

## NeoBundle

Add this to your `.vimrc`:

```vim
NeoBundle 'mtahmed/click.vim'
```
